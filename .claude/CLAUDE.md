# VA Hair

Website for a Virginia-based hair salon/stylist business.

## Tech Stack
- Astro 5, Tailwind CSS 4
- Vercel hosting

## User Preferences

### Workflow
- Autonomous execution, parallel agents when possible
- Haiku/Sonnet only (no Opus)

### Code Standards
- Immutability enforced, 800 line max
- No console.log in committed code
- No hardcoded values
